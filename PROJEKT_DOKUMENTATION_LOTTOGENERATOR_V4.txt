PROJEKT: Lottogenerator V4
=========================

1. ÜBERBLICK
------------
Projektname: lottogenerator_v4

Ziel:
Flutter-App zur Verwaltung, Analyse und Generierung von Lottozahlen
für:
- Lotto 6aus49
- Eurojackpot

Eigenschaften:
- Offline-fähig
- Lokale SQLite-Datenbank
- Historische Ziehungen importierbar
- Erweiterbar für Statistik und Simulation

Primäre Zielplattform: Android
Build-System: Codemagic
Entwicklungsumgebung: Termux + Debian


2. TECHNOLOGIESTACK
-------------------
- Flutter (Dart SDK >= 3.0)
- sqflite (SQLite lokal)
- path
- http / html (für frühere Scraper)
- Keine Cloud-Abhängigkeit
- Keine externen APIs im Produktivbetrieb

4. WICHTIGE PFADE
-----------------
Assets:
- assets/data/lotto_6aus49.json
- assets/data/eurojackpot_2012_2025.txt

Datenbank:
- SQLite (automatisch erzeugt)
- Tabelle: ziehungen


5. DATENBANKMODELL
------------------
Tabelle: ziehungen

Felder:
- id INTEGER PRIMARY KEY
- datum TEXT (ISO-Format)
- spieltyp TEXT ('6aus49' | 'eurojackpot')
- zahlen TEXT (CSV-String der Zahlen)
- superzahl INTEGER
  - Lotto 6aus49: echte Superzahl
  - Eurojackpot: immer 0


6. IMPORTLOGIK
--------------
Lotto 6aus49:
- Quelle: lotto_6aus49.json
- 6 Zahlen + Superzahl
- Gruppierung nach Datum
- Dublettenprüfung vor Insert

Eurojackpot:
- Quelle: eurojackpot_2012_2025.txt
- Format:
  YYYY-MM-DD | n n n n n | e e
- Speicherung:
  zahlen = 7 Werte (5 + 2)
  superzahl = 0
